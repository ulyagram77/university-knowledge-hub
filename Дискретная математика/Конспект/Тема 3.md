`Размещение` с повторениями:
$$\tilde{A}_{n}^r = n^r$$

`Размещение` без повторений:
$$A_{n}^r = \frac{n!}{(n-r)!}$$

`Перестановка`:
$$P_{n} = \frac{n!}{(n-n)!} = \frac{n!}{0!} = n!$$

`Комбинации` без повторений:
$$C_{m}^n = \frac{m!}{(m-n)!\cdot n!}$$

`Комбинации` с повторениями:
$$C_{n+m-1}^m = \frac{(n+m-1)!}{m! \cdot (n-1)!}$$
